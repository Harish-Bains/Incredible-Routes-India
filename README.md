## 🚀 Incredible Routes India

«A Flask-based Travel Analytics & Budget Prediction System built using Python, Flask, Power BI, SQL, HTML, CSS, and JavaScript.»

---

## 📌 Project Overview

Incredible Routes India is a data-driven travel analytics platform that helps users explore tourist destinations across India through interactive dashboards, travel insights, and budget estimation. The application combines data analysis, route calculation, and visualization to provide meaningful information for travelers and analysts.

The project integrates Power BI dashboards with a Flask web application to present tourism data such as destination popularity, hotel ratings, seasonal trends, transportation costs, and estimated travel budgets.

---

##  ✨ Key Features

- 🗺️ Travel Destination Analysis

- 🌦️ Seasonal Tourism Analysis
- 🚆 Transportation Cost Estimation

Clone the Repository

Navigate to Project

cd Incredible-Routes-India

Install Dependencies

pip install -r requirements.txt
